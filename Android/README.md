# VigyBag Android



## Installation
1. Clone the parent repository

    ```bash
    git clone https://github.com/codervivek5/VigyBag.git
    ```
2. Inside Android Studio, navigate to app code as follows: 
    ```bash
    File -> Open -> (Parent folder of the whole repository)/VigyBag/Android
    ```

## Generating signed APK
From Android Studio:
1. ***Build*** menu
2. ***Generate Signed APK...***
